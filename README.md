# 🏥 Patient Appointment Scheduling System Using Queue

A web-based Patient Appointment Scheduling System developed using **Python (Flask)** that implements **Queue (FIFO)** and **Priority Queue** data structures to efficiently manage patient appointments. The system helps reduce waiting time, streamline appointment scheduling, and prioritize emergency patients.

---

## ✨ Features

- 👤 Patient Registration & Login
- 📅 Book Patient Appointments
- 🩺 Doctor Dashboard
- 📋 Queue-Based Appointment Management (FIFO)
- 🚨 Emergency Patient Priority Queue
- 📆 Multi-Day Appointment Scheduling
- 💳 Payment Module
- 🔄 Follow-Up Appointment Booking
- 📱 Responsive and User-Friendly Interface

---

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python (Flask)
- **Database:** SQLite
- **Data Structures:** Queue (FIFO), Priority Queue

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/vasanthasrisala/Queue-Based-Appointment-System.git
```

### Navigate to the project

```bash
cd Queue-Based-Appointment-System
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
